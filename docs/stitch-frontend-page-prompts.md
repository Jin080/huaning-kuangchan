# 华宁矿产竞拍平台 Stitch 前端页面生成提示词

> 用途：将 PRD 中的页面方案转成可直接复制给 Stitch 的中文 Prompt，用于生成 PC 门户、PC 管理后台、企业用户中心页面。  
> 范围：第一批重点生成 PC 门户和 PC 管理后台；微信小程序端仅做页面规划，不展开 UI。  
> 注意：以下 Prompt 只要求生成前端页面，不包含后端代码，不生成接口，不编造 PRD 外业务功能。

## 1. 全局设计要求

请生成“华宁矿产竞拍平台”的前端页面 UI。视觉参考政务交易平台、矿产交易平台、公共资源交易平台，整体风格稳重、可信、清晰、偏政务业务系统。

通用要求：

- PC 门户适合信息展示、公告浏览、竞价入口。
- PC 管理后台适合表格、筛选、审核、状态流转。
- 首页可以有门户感，但不要做互联网营销风首页。
- 后台必须是高效率管理系统风格。
- 不要过度渐变，不要花哨动效。
- 不要编造 PRD 外的业务功能。
- 不要生成后端代码。
- 不要生成接口逻辑。
- 使用真实业务字段占位，不使用无关营销文案。

推荐视觉：

- 主色：政务蓝、深灰、白色、浅灰。
- 强调色：竞价金额和风险操作可使用克制的红色或橙色。
- 组件：顶部导航、左侧菜单、面包屑、筛选表单、表格、状态标签、详情抽屉、确认弹窗、数据卡片。
- 页面密度：门户中等密度，后台高信息密度。

## 3. PC 门户页面 Prompt

### 3.1 首页

```text
请生成“华宁矿产竞拍平台”PC 门户首页。

页面用途：
用于展示平台入口、数据看板、即将拍卖公告、正在竞价标的、成交公示、矿产资源信息和信息资讯。

视觉风格：
参考政务交易平台、矿产交易平台、公共资源交易平台。整体稳重、可信、清晰。首页可以有门户感，但不要做互联网营销风首页，不要过度渐变，不要花哨动效。

布局结构：
顶部为门户导航栏，左侧显示平台名称“华宁矿产竞拍平台”，中间导航包括：首页、矿产资源、即将拍卖、正在竞价、成交公示、信息资讯、公开说明；右侧放搜索入口、登录、企业入驻。

首屏区域：
展示平台名称、搜索框、公告快捷入口。整体像政务门户，不要像商业落地页。

主要模块：
1. 数据看板：本年成交量、本年成交额、累计成交量、累计成交额。
2. 即将拍卖公告：展示公示期拍品。
3. 正在竞价：展示竞拍期拍品和倒计时。
4. 成交公示：展示成交拍品、中标企业、最终成交价。
5. 矿产资源：用小卡片展示拍品标题、单价/起拍价、数量、供应商、产地、简要商品信息。
6. 信息资讯：展示政策法规、交易公告、矿能动态。
7. 公开说明入口。

按钮：
登录、企业入驻、搜索、查看更多、查看详情、进入竞价。

交互状态：
加载时展示骨架屏；无数据时模块内展示“暂无相关数据”；未登录用户点击企业操作时引导登录。

限制：
只生成前端静态页面设计，不包含后端代码，不生成接口，不编造 PRD 外业务功能。
```
